WHAT WILL YOU LEARN

CORE ENGINEERING SKILLS

Logical thinking, problem solving, debugging, these require training and practice. They don’t come naturally to most of us. We’ll make sure that you don’t just learn what to do, but how to think.

FRONT END DEVELOPMENT

This ain’t your uncle’s Javascript class. We’re going above and beyond basic HTML/CSS and Javascript to teach you advanced methods that are used in the industry today. Mentors that are on the bleeding edge of front-end development will help you along the way.

GIT COLLABORATION AND WORKFLOW

Git has become the de facto standard for source version control and team collaboration. Git has its own jargon, best practices, and workflows, enough to keep you busy for a while. Yet these tactics will be crucial in your contribution in the workplace and in securing, protecting and archiving your code.

WEB APPLICATION BASICS

Building static “websites” is a thing of the past. Big brands and modern web products include a variety of front-end technologies to provide a rich user experience. We’ll teach you the basic technologies and methods used in the entire stack to build web applications that users love.

ADV. JAVASCRIPT & FRAMEWORKS

Javascript has come a long way in the past 18 years, and developers who wield the latest and greatest tools will be best suited to solve today’s web technology problems. Javascript frameworks have greatly simplified the complex task of creating a responsive, asynchronous web application. You’ll be ahead of the learning curve.

TEST DRIVEN DEVELOPMENT

Writing buggy code is no fun, and unless you’re a robot you’re going to have some bugs in your code. We’ll teach you the latest techniques in TDD (Test Driven Development) to ensure your code is strong and stable, and that one edit to a single line won’t bring down your entire application.

DEVELOPMENT BEST PRACTICES

Web development is a craft, not a skill. There are tools of the trade and tricks to master. Beyond knowing just programming, there is an entire ecosystem of communities of practice, tools to make you more productive, and attitudes that will help you be successful. You’ll become fluent in development-ese.

NODE.JS AND METEOR

We want to give you all of the necessary tools to make a full web application. That includes server side components that will allow you to store, retrieve, and manipulate data for your users. You’ll know enough to be dangerous.
